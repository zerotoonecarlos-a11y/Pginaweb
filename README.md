# Projeto de Exemplo de Página Web

Este projeto é um exemplo simples de uma página web estática que simula a aparência de uma página de canal do YouTube. Ele foi criado para fins educacionais e de demonstração, utilizando apenas HTML e CSS.

## Estrutura do Projeto

O projeto está organizado da seguinte forma:

- `index.html`: Contém a estrutura principal da página web.
- `estilo.css`: Contém as regras de estilo para a página.
- `assets/`: Contém os recursos visuais utilizados na página, como ícones e imagens.
  - `icons/`: Armazena os ícones em formato SVG.
  - `images/`: Armazena as imagens utilizadas no layout.

## Funcionalidades

A página é puramente visual e não possui funcionalidades interativas. Ela demonstra o uso de:

- HTML semântico para estruturar o conteúdo.
- CSS para estilização, incluindo layout com Flexbox, cores, fontes e imagens de fundo.
- Media queries para responsividade básica.

## Como Utilizar

Para visualizar a página, basta abrir o arquivo `index.html` em qualquer navegador de internet. Nenhuma instalação ou configuração adicional é necessária.

## Observações

Este é um projeto de front-end básico e não inclui nenhum back-end ou interatividade com JavaScript. As imagens e ícones são meramente ilustrativos.
